### 1. 基本初等函数与幂函数

$$(C)' = 0,\quad (x^n)' = nx^{n-1},\quad (\sqrt{x})' = \frac{1}{2\sqrt{x}},\quad \left(\frac{1}{x}\right)' = -\frac{1}{x^2}$$

### 2. 指数与对数函数

$$(e^x)' = e^x,\quad (a^x)' = a^x \ln a,\quad (\ln x)' = \frac{1}{x},\quad (\log_a x)' = \frac{1}{x \ln a}$$

### 3. 三角函数

$$(\sin x)' = \cos x,\quad (\cos x)' = -\sin x,\quad (\tan x)' = \sec^2 x,\quad (\cot x)' = -\csc^2 x$$

$$(\sec x)' = \sec x \tan x,\quad (\csc x)' = -\csc x \cot x$$

### 4. 反三角函数

$$(\arcsin x)' = \frac{1}{\sqrt{1-x^2}},\quad (\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$$

$$(\arctan x)' = \frac{1}{1+x^2},\quad (\text{arccot } x)' = -\frac{1}{1+x^2}$$
