### 1. 基本幂函数与指数函数

$$\int k \, dx = kx + C,\quad \int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$

$$\int \frac{1}{x} \, dx = \ln|x| + C,\quad \int \frac{1}{x^2} \, dx = -\frac{1}{x} + C,\quad \int \frac{1}{\sqrt{x}} \, dx = 2\sqrt{x} + C$$

$$\int e^x \, dx = e^x + C,\quad \int a^x \, dx = \frac{a^x}{\ln a} + C$$

### 2. 三角函数 (基础)

这里是导数公式的逆运算：

$$\int \sin x \, dx = -\cos x + C,\quad \int \cos x \, dx = \sin x + C$$

$$\int \sec^2 x \, dx = \tan x + C,\quad \int \csc^2 x \, dx = -\cot x + C$$

$$\int \sec x \tan x \, dx = \sec x + C,\quad \int \csc x \cot x \, dx = -\csc x + C$$

### 3. 三角函数 (进阶 - 常用且易忘)

这组公式在解题时非常关键，尤其是 $\sec x$ 的积分：

$$\int \tan x \, dx = -\ln|\cos x| + C,\quad \int \cot x \, dx = \ln|\sin x| + C$$

$$\int \sec x \, dx = \ln|\sec x + \tan x| + C$$

$$\int \csc x \, dx = \ln|\csc x - \cot x| + C$$

### 4. 对应反三角函数的积分

这里给出了带有参数 $a$ 的通用形式，比单纯的 $x$ 更实用：

$$\int \frac{1}{\sqrt{a^2 - x^2}} \, dx = \arcsin \frac{x}{a} + C$$

$$\int \frac{1}{a^2 + x^2} \, dx = \frac{1}{a} \arctan \frac{x}{a} + C$$

$$\int \frac{1}{x\sqrt{x^2 - a^2}} \, dx = \frac{1}{a} \text{arcsec} \frac{|x|}{a} + C$$
