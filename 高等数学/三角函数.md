### 1. 同角三角函数关系 (倒数、商、平方)

#### 1. 倒数关系 (Reciprocal Identities)

这是 $\sec x$ 和 $\csc x$ 最本质的定义：

$$\sec x = \frac{1}{\cos x},\quad \csc x = \frac{1}{\sin x}$$

$$\sec x \cdot \cos x = 1,\quad \csc x \cdot \sin x = 1$$

#### 2. 商数关系 (Quotient Identities)

除了 $\tan x = \sin x / \cos x$ 之外，$\sec x$ 和 $\csc x$ 也有类似的关系：

$$\frac{\sin x}{\cos x} = \tan x,\quad \frac{\cos x}{\sin x} = \cot x$$

$$\frac{\sec x}{\csc x} = \tan x,\quad \frac{\csc x}{\sec x} = \cot x$$

#### 3. 平方关系 (Pythagorean Identities)

这一组公式在积分换元（三角代换）中通过 $\sec x$ 去掉根号时非常关键：

$$\sin^2 x + \cos^2 x = 1$$

$$1 + \tan^2 x = \sec^2 x,\quad \tan^2 x = \sec^2 x - 1$$

$$1 + \cot^2 x = \csc^2 x,\quad \cot^2 x = \csc^2 x - 1$$
### 2. 二倍角公式 (Double Angle Formulas)

这是在积分和求导中最常用来化简的公式：

$$\sin 2x = 2\sin x \cos x,\quad \tan 2x = \frac{2\tan x}{1-\tan^2 x}$$

$$\cos 2x = \cos^2 x - \sin^2 x,\quad \cos 2x = 2\cos^2 x - 1,\quad \cos 2x = 1 - 2\sin^2 x$$

### 3. 降幂公式 (用于积分)

由二倍角公式推导而来，处理 $\sin^2 x$ 和 $\cos^2 x$ 的积分时必用：

$$\sin^2 x = \frac{1 - \cos 2x}{2},\quad \cos^2 x = \frac{1 + \cos 2x}{2}$$

### 4. 和差角公式 (Sum and Difference Formulas)

$$\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$$

$$\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$$

$$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$$
